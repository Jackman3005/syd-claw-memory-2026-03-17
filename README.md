# SydClaw Talk — Memory Systems & Building Long-Term Context

Memory talk for the Sydney OpenClaw Meetup, March 2026.

### 📺 [View the slides](https://jackman3005.github.io/syd-claw-memory-2026-03-17/)

## Setup

```bash
bun install
```

## Development

```bash
bun run dev
```

Opens a dev server at `http://localhost:3000` with hot reload. Edit files in `src/` and the browser auto-refreshes.

## Build & Deploy

```bash
bun run build    # Build to dist/
bun run deploy   # Deploy to GitHub Pages
```
